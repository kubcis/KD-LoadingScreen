# 🚀 Clean, Spotify inspired Loading Screen

A fully customizable, optimized, and modern loading screen for FiveM servers, created by **Kubciis Development**. 

## ✨ Features
* **Dynamic Content:** Easily configure your server news, team members, and links.
* **Advanced Music Player:** Volume slider, song progress tracking (clickable), and smart track playlist.
* **Animated Feedback:** Smooth top-down toast notifications when copying admin handles or opening links.
* **Background Slider:** Seamlessly cycles through randomized background images.

## 🛠️ Installation
1. Download this repository.
2. Drop the folder into your FiveM server's `resources` directory.
3. Add `ensure kd_loadingscreen` to your `server.cfg`.
4. Open `script.js` and edit the content to match your server details.
